# KB_SkeletonProject_21-5

## 실행 방법

1. 프로젝트를 다운로드합니다.

```sh
git clone -b example2 --single-branch https://github.com/Kimyeongmok/KB_SkeletonProject_21-5.git
```

2. 다운로드한 폴더로 이동합니다.

```sh
cd KB_SkeletonProject_21-5
```

3. 패키지를 설치합니다.

```sh
npm install
```

4. JSON 서버를 실행합니다.

```sh
npm run server
```

5. 새 터미널을 열고 같은 폴더에서 개발 서버를 실행합니다.

```sh
npm run dev
```
