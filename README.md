## Description

본 프로젝트의 주제는 가계부 서비스 앱이다. 수업시간에 배운 기술 요소를 실제 서비스에 접목하여 각 개인의 프로그래밍 능력 향상을 목표로 한다.

기능 요구사항으로는 1. 수입/지출 기록 2. 거래내역 조회 3. 월별 재정 요약 4.데이터저장 이 있다.
